VizLang Studio reads a LangGraph .jsonl trace and renders the run as a left-to-right node graph. The active edge animates, tool calls expand inline, and retries stack — the cursor moves through time, the view does not.

Three rules: every node has a stable identity across runs, every edge carries the message that traversed it, and the panel never autoscrolls. Read the field note →
